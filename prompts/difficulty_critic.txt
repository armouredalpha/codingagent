EASY

Requirements:

* Bloom level: Remember, Understand, or Apply
* Single primary concept
* Single file modification
* Less than 20 LOC expected
* Minimal debugging
* No architecture decisions
* No integration work

Typical Examples:

* Publisher completion
* Subscriber callback completion
* Parameter update
* Topic correction

Target Role:

Robotics Intern

Expected Solve Time:

5-20 minutes

---

MEDIUM

Requirements:

* Bloom level: Apply or Analyze
* 2-3 related concepts
* 1-2 files modified
* 20-80 LOC expected
* Moderate debugging
* Some integration required
* Candidate must understand ROS interactions

Typical Examples:

* Launch file debugging
* TF repair
* Service-client interaction
* Sensor processing logic
* Multi-node communication

Target Role:

Junior Robotics Engineer

Expected Solve Time:

20-60 minutes

---

HARD

Requirements:

* Bloom level: Evaluate or Create
* Multiple concepts combined
* Multiple files may be modified
* More than 80 LOC expected
* Integration-heavy workflow
* System-level reasoning required
* Edge cases or reactive logic required
* Candidate must design part of the solution

Typical Examples:

* Gazebo + Sensor Plugin Integration
* TF + Launch + Parameters
* Multi-node robotic workflow
* Full simulation pipeline
* Robot behavior architecture

Target Role:

Robotics Engineer

Expected Solve Time:

60-180 minutes
